# EP02 发布文案

## 标题

OneNote 也能完整渲染 Markdown？标题、任务、代码高亮、LaTeX 实测

## 简介

用同一份真实测试文档，完整演示 OneNote Markdown 插件的渲染效果：

- H1-H6 标题层级
- 粗体、斜体、删除线、高亮、下划线和行内代码
- 无序列表、有序列表与任务列表
- 引用块
- C# 代码语法高亮
- 行内与块级 LaTeX 公式
- 原始 Markdown 与渲染结果前后对比

项目开源地址：https://github.com/oldding/OneNote-markdown

国内下载：https://nexanote.cn/
