# EP04 配音稿：导入、导出与个性化设置

1. 开场：导入、导出、复制与设置总览。
2. GitHub README 需要先下载或克隆为本地 `.md`，当前版本不直接读取 URL。
3. 导入使用 UTF-8，渲染结果追加到当前页面，不覆盖原页面。
4. 导出把当前 OneNote 页面转换为 UTF-8 `.md`，默认文件名来自页面标题。
5. “复制 Markdown”与 `F8` 都会把转换结果写入系统剪贴板。
6. 字体和字号可分别设置；保存后重新渲染，重要页面建议重启 OneNote 后再最终渲染。
7. 代码行号适合教程和审阅，可按需开关。
8. 语言支持自动、中文和 English；完整刷新建议重启 OneNote。
9. 总结与 GitHub、国内下载地址。
