# EP04 发布文案

## 标题

OneNote Markdown 导入导出全流程：README、.md、复制与个性化设置

## 简介

本期演示 OneNote Markdown 的内容迁移与个性化能力：

- 将 GitHub 仓库的 `README.md` 下载到本地后导入 OneNote
- 将当前页面导出为 UTF-8 `.md` 文件
- 使用“复制 Markdown”或 `F8` 写入剪贴板
- 设置正文字体、代码字体、数学字体
- 分别调整正文与代码字号
- 开启或关闭代码行号
- 在自动、中文和 English 之间切换界面语言

当前版本导入的是本地 `.md` / `.markdown` 文件，不直接读取 GitHub URL。
导入结果会追加到当前页面，不会覆盖原内容。语言切换要完整刷新功能区时，
建议重启 OneNote。

项目开源地址：https://github.com/oldding/OneNote-markdown

国内下载：https://nexanote.cn/

## 置顶评论

三个最常用动作：导入本地 README，导出 `.md`，按 `F8` 复制 Markdown。
设置修改后重新渲染；语言切换完整生效建议重启 OneNote。
