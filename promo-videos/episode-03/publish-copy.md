# EP03 发布文案

## 标题

OneNote Markdown 实时模式实测：F5、F8、Ctrl+\ 到底怎么用？

## 简介

本期实测 OneNote Markdown 的实时模式和效率快捷键：

- 回车前的 Markdown 源码状态
- 回车后的 OneNote 格式化结果
- OneNote 编辑提交带来的刷新延迟
- `F5`：整页渲染并追加结果
- `F8`：复制当前页 Markdown
- `Ctrl+\`：开启或关闭实时模式

重要说明：当前发布版并非左右双栏预览，而是在当前内容块中输入源码，回车提交后原位渲染。刷新受 OneNote 提交时机影响，通常会有短暂等待。

项目开源地址：https://github.com/oldding/OneNote-markdown

国内下载：https://nexanote.cn/
